# Шабашер — приложение для организации корпоративных мероприятий

## 1. Описание идеи

*Шабашер* — это мобильное приложение для автоматизации организации корпоративных мероприятий в малом и среднем бизнесе.  
Приложение объединяет планирование, координацию и вовлечение сотрудников в едином пространстве, делая процесс организации событий прозрачным, удобным и интерактивным.

## 2. Тип проекта

Мобильное приложение.

## 3. Цель проекта и задачи

*Цель:*  
Сократить трудозатраты на организацию корпоративов и тимбилдингов за счёт автоматизации и повышения вовлечённости участников.

*Задачи:*
- Разработать интуитивно понятный интерфейс для создания и управления событиями.
- Реализовать систему приглашений через QR-коды и ссылки.
- Внедрить инструменты коллаборативного планирования: сбор идей, сбор средств, управление статусами участников.
- Добавить встроенные мини-игры для повышения вовлечённости.
- Обеспечить стабильную работу клиент-серверного взаимодействия.

## 4. Структурные части проекта и технологии

*Frontend:*
- Язык программирования: Kotlin
- Фреймворк/Платформа: Android SDK, Jetpack Compose

*Backend:*
- Язык программирования: C#
- Фреймворк: ASP.NET Core

*База данных:*
- PostgreSQL
